== Description ==
Admin Notes Manager is a lightweight plugin that allows administrators to manage internal notes directly from the WordPress admin panel. This plugin enables adding, deleting, and exporting admin notes efficiently.

== Features ==
* Create, delete, and manage admin notes
* Export notes as CSV
* Clean and simple UI

== Installation ==
1. Upload the `admin-notes-manager` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'Admin Notes' in the WordPress dashboard.

== Frequently Asked Questions ==
= Can I edit a note after adding it? =
Currently, you can add and delete notes, but an edit function will be available in future updates.

= Where are notes stored? =
Notes are stored in the WordPress options table for quick access.
